# Bot Discord - World Blox

Bot para gerenciar vendas de contas Blox Fruits.

## Configuração local
1. Instale Python 3.10 ou superior.
2. Abra o terminal na pasta do projeto.
3. Instale dependências:
   pip install -r requirements.txt
4. Configure a variável de ambiente `TOKEN` com o token do Discord.
5. Execute:
   python bot.py
